Torrent
=======

Torrent finder for movies and TV series by IMDB on YTS, RARBG and The Pirate Bay.

## Usage

### Installation

#### `go get`

```shell
$ go get -u -v github.com/jelliflix/torrent
```

#### `go mod` (Recommended)

```go
import "github.com/jelliflix/torrent"
```

```shell
$ go mod tidy
```
